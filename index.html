1. La narrativa del proyecto a realizar e insertar una imagen alusiva.

En la era digital actual, tener una presencia en línea es esencial para cualquier empresa que intente llegar a su audiencia de manera efectiva. En este contexto, la necesidad de crear una página web para una tienda de cupcakes no solo es algo práctico, sino también una oportunidad para transmitir la esencia y el encanto de la marca a través de la web. En este trabajo final, analizaremos el proceso de concepción, diseño y desarrollo de la página web de "Dulce Tentación", una tienda de cupcakes que cautiva los sentidos con sus deliciosos productos y encanta a sus clientes con su enfoque creativo y apasionado.

Analizaremos cada paso del proceso, desde la planificación inicial hasta la implementación técnica, con el objetivo de comprender cómo la presencia en línea puede impulsar el crecimiento y la visibilidad de este pequeño negocio. Examinaremos cómo cada decisión, desde la elección de la plataforma de desarrollo hasta la selección de colores y diseño, estuvo cuidadosamente alineada con la identidad y los valores de la marca, convirtiendo a la página web en una poderosa herramienta para atraer y cautivar a los amantes de los cupcakes.

3. Cuadro:
Caso Ver Menu de Postes
Actores Cliente: Usuario que visita la página web para ver el menú de postres y decidir qué postres le interesan.
Objetivos asociados Mejorar la experiencia de usuario y aumentar las conversiones mediante la optimización de las herramientas de búsqueda y filtrado/td>
Descripción Este caso de uso permite a los usuarios explorar el menú de postres ofrecidos por la tienda de manera detallada, incluyendo información sobre cada postre, precios, descripciones y posiblemente imágenes.
Precondición La página web está disponible y funcionando correctamente.
La información del menú de postres está actualizada y correcta.
Secuencia Normal Paso
Accion
Actor Sistema
1 El cliente accede a la página web de la tienda de postres. El sistema muestra la interfaz principal con todas las opciones disponibles para el cliente.
2 El cliente busca y selecciona la sección del menú de postres. El sistema muestra una lista de postres disponibles con sus nombres, precios y breves descripciones.
3 El cliente puede hacer clic en un postre para ver más detalles como ingredientes, tamaño, opciones de personalización, etc. El sistema muestra los detalles completos del postre seleccionado.
Escenario Alternativo Paso
Accion
Actor Sistema
1 Si el cliente no encuentra un postre específico, puede utilizar la barra de búsqueda para buscarlo por nombre o categoría. El sistema muestra un mensaje indicando al cliente que utilice la barra de búsqueda para encontrar el postre deseado. Se le anima a buscar por nombre o categoría para facilitar la búsqueda.
2 Si el cliente desea filtrar los postres por precio, tipo (tarta, pastel, helado, etc.) u otros criterios, puede utilizar las opciones de filtrado proporcionadas en la página. El sistema proporciona instrucciones al cliente sobre cómo utilizar los filtros disponibles para refinar su búsqueda. Se le indica que puede filtrar por precio, tipo de postre y otros criterios para encontrar exactamente lo que desea.
Postcondición El cliente ha visto el menú de postres y puede tomar decisiones informadas sobre qué postres le gustaría ordenar o investigar más a fondo.
